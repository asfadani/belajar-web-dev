# Proyek Eksplorasi Web Dev

Repositori ini adalah *sandbox* (tempat uji coba) untuk mempelajari fundamental pengembangan web sebelum membangun *landing page* utama.

**Teknologi yang digunakan saat ini:**
* HTML5
* Tailwind CSS (via CDN)
* Vanilla JavaScript

**Rencana Fase Selanjutnya:**
* Backend menggunakan Golang
* Database ringan menggunakan SQLite
